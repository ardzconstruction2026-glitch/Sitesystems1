📋 CSV Format Guide
Standard columns (all element types):
type · mark · length · width · height · unit

Beam with coordinates (length auto-computed):
type · mark · x1 · y1 · x2 · y2 · width · height
Coordinates always in meters. Length = √((x2-x1)²+(y2-y1)²)

Rules:
• First row must be the header row
• type must be one of: Beam, Column, Footing, Slab, Wall, Other
• unit applies to width and height only when using coords (length is always meters)
• unit must be one of: m, cm, mm
• Dimensions must be positive numbers · mark must not be empty

Example — standard:
type,mark,length,width,height,unit Beam,B-1,6,0.3,0.45,m Column,C-1,3,0.4,0.4,m Footing,F-1,1.5,1.5,0.45,m
Example — beam with coordinates:
type,mark,x1,y1,x2,y2,width,height Beam,B-1,0,0,6,0,0.3,0.45 Beam,B-2,6,0,6,4.5,0.3,0.45 Beam,B-3,0,4.5,6,4.5,0.3,0.45
